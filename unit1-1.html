Unit 1.1: Grammar

The Metalanguage of Language
Home > Unit 1 > Unit 1.1

In order to learn to read Greek, you must first develop a strong grammatical foundation. If you are going to talk about a language, you need to know the language of language. In particular, this course will be using grammatical trees built on the rules of something called Dependency Syntax, a simple framework for discussing the structure of sentences. These lessons will teach you how to talk about language, using English for all the examples.
